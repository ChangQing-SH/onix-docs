Version Compatibility¶

Attention

The hardware in the following release groups have additional, inter-group compatibiities. Comprehensive ONIX hardware version compatibility is documented on this google sheet.

Beta Test¶

The following hardware components comprise our beta-test system. These hardware revisions are compatible:

  1. headstage-neuropix v1.2

  2. headstage-64 v1.3

  3. eib-64 v1.2

  4. eib-64-large v1.0

  5. omnetics-adapter-64 v1.0

  6. test-board-64 v1.2

  7. breakout v1.3

  8. fmc-host v1.3
