nano-z-adapter-64 v1.0¶

nano-z-adapter-64 v1.0 is an adapter board that allows ONIX electrode interface and adapter boards to be used with the nanoZ electrode impedance tester and plating device. This adapter is compatible with:

  • eib-64 v1.2

  • eib-64-large v1.0

  • omnetics-adapter-64 v1.0
