The ONIX APIs are high-performance software interface for host computer interaction with ONIX hardware. API implementations are compliant with the Open Neuro Interface Specification. These APIs are simple and has minimal external dependencies. Therefore, they are aimed at the creation of higher level language bindings and/or integration into existing acqusition software.

  • liboni is an ANSI-C implementation of the ONI API Specificaiton It contains functions for configuring hardware, streaming data to and from hardware, and controlling hardware during operation.

  • cpponi are C++14 bindings for liboni.

  • clroni are CLR/.NET bindings for liboni.

  • ONIX Driver Implementations are ONIX device driver implementations
